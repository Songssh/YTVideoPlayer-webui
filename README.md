# YTVideoPlayer-webui
YTVideo player

## Requirements
- Python

## Setup
1. Instsall youtube-dl
2. Create environment
```
# anaconda
conda create -n ytdl python=3.9.13
```
```
# Venv
python -m venv ytdl
```
3. Activate environment
```
# anaconda
activate ytdl
```
```
# Venv
source ytdl/Scripts/activate
```
4. Install python requirements
```
pip install -r requirements.txt
```

## How to use
```
python main.py
```
